# ADR-0004: Phase 0 kill date — 2026-08-12

**Date:** 2026-07-12
**Status:** Accepted

## Context
Jordi runs multiple active ventures (Mc Fly, NEXO, Tiny Houses) and has a known pattern of
accumulating projects. An R&D project without a deadline becomes an infinite hobby.

## Decision
Hard kill date: **2026-08-12** (one month from kickoff). If no promising, cost-adjusted bias
has appeared in the data by then, the project is archived without guilt via KILL.md.

## Alternatives considered
- No deadline, "until we know" — rejected: silently cannibalizes the businesses that pay the bills.
- Effort budget (hours/week) instead of a date — not rejected, just undefined; can be added as a complement (open item in STATUS.md).

## Consequences
Scope pressure is a feature: Phase 0 must fit in one month, which forces the historical-first
approach (ADR-0003). Archiving ≠ failure; a documented negative result is a valid outcome.
